# **DOGS** | Proyecto Individual 🐶

## Sobre el proyecto

¡Bienvenidos al proyecto individual aplicación canina!

En este proyecto, se ha empleado las tecnologías de React, Redux, Node, Express y Sequelize para crear una aplicación de página única que ofrece a los usuarios la capacidad de buscar, visualizar, filtrar, ordenar y crear información relacionada con perros.
Este logro se materializó mediante la integración de funcionalidades que posibilitan el acceso a datos desde una API externa, así como la consulta y gestión de información almacenada en nuestra base de datos interna.
